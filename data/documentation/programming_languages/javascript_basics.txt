JavaScript is a dynamic programming language primarily used for web development.

Variables:
let age = 20
const name = "John"
var oldStyle = true

Data Types:
Number
String
Boolean
Object
Array
Null
Undefined

Functions:
function add(a, b){
return a + b;
}

Arrow Functions:
const add = (a,b) => a+b;

Arrays:
push()
pop()
map()
filter()
reduce()

Objects:
const user = {
name: "John",
age: 20
}

DOM Manipulation:
document.querySelector()
document.getElementById()

Async Programming:
Promises
async/await
fetch()
