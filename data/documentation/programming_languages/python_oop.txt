Class:
A blueprint for creating objects.

Object:
An instance of a class.

Example:
class Car:
def init(self, brand):
self.brand = brand

def drive(self):
    return "Driving"

Inheritance:
Allows a class to inherit properties and methods from another class.

class ElectricCar(Car):
pass

Encapsulation:
Restricting direct access to data using methods and private attributes.

Polymorphism:
Different classes can implement methods with the same name.

Abstraction:
Hiding implementation details and exposing only essential functionality.
